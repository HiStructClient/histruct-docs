# 🔧 Přizpůsobení HiStruct pro váš produkt

Instalační systém pro střešní doplňky, oplechování a okapový systém je nastaven samostatně pro každého konkrétního výrobce v separátním konfiguračním souboru. Obsahuje informace o tom, jak má být střešní plocha osazena doplňky, možné barevné kombinace, informace o výkazu materiálu (čísla dílů) a další.
Tyto konfigurační soubory se vytvářejí v rámci projektu přizpůsobení, během kterého upravujeme konfigurátor pro výrobce podle jeho konkrétního produktového portfolia.

## Co přizpůsobujeme?

- Konfigurační soubory pro dostupné střešní krytiny, oplechování a okapový systém
- Geometrie jednotlivých komponent, textury, mapy (pokud je třeba)
- Nastavení pro úspěšné vygenerování výkazu materiálu z dostupné geometrie, obecné prvky bez geometrické reprezentace, upevňovací prvky, podkladní vrstvy atd.
- Výstupy dle požadavků výrobce: výkazy materiálu, nabídky, výkresy, pohledy, exporty atd.
- Napojení na ceník přes API do stávajícího systému výrobce
- Napojení do ekosystému výrobce, exporty nabídek, propojení s ERP a další dle požadavků
