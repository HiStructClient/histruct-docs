# Výkresy
Menu <u><i>Výkresy</i></u> umožňuje vytvořit a dále editovat výkresy střešní konstrukce. 

U všech výkresů je po otevření možné provádět další editace:
* Měnit jejich měřítko.
* Přidávat čáry, kóty nebo popisky.
<style>
h2{
  border-bottom: none;
  margin-top: 10px;
  margin-bottom: 0px;
}
p{
  border-bottom: none;
  margin-top: 10px;
  margin-bottom: 10px;
}
</style>
