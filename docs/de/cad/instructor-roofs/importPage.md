# Import

Die Schaltfläche *Importieren* dient zum einfachen Importieren verschiedener Unterlagen, die Ihnen das Modellieren des gewünschten Projekts erleichtern. Sie können Dateien im DXF- oder OBJ-Format einfügen. Außerdem haben Sie die Möglichkeit, Ihre Unterlagen aus dem PDF-Format in einen der unterstützten Dateitypen zu konvertieren, was Ihnen neue Möglichkeiten eröffnet und die Arbeit mit dem Modellieren erleichtert.
