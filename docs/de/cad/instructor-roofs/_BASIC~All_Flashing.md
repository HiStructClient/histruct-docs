# Leisten

<p>Ermöglicht das Ändern der Farbe einzelner Leisten-Elemente.</p>

<p>Die Farbe kann mit den Schaltflächen auf der linken Seite für alle Leisten-Elemente einer bestimmten Gruppe angepasst werden, oder Sie können direkt auf die einzelnen Elemente im Modell klicken und deren Farbe individuell anpassen.</p>

<p><b><u>Die Anzahl der editierbaren Leisten-Gruppen kann je nach Möglichkeiten der einzelnen Modelle variieren.</u></b></p>

<hr class="main">

<h2>Bearbeiten von Elementgruppen mit den Schaltflächen links</h2>
<p>Durch Klicken auf die Leisten-Schaltfläche auf der linken Seite können die Eigenschaften der gesamten entsprechenden Leisten-Gruppe bearbeitet werden.</p>

{{ box_icons([
  ("img/ColorsIcon64x64.png", "Leisten"),
  ("img/ColorsIcon64x64.png", "Fallrohr"),
], "... und mehr", width=55, height=55, fontSize=10) }}

<p>Die vorgenommenen Eigenschaftsänderungen werden auf alle Elemente der angegebenen Gruppe angewendet.</p>

<hr class="main">

<h2>Bearbeiten einzelner Elemente</h2>
<p>Die Eigenschaften einzelner Elemente können direkt durch Klicken auf die jeweiligen Elemente im Modell geändert werden.</p>

<hr class="main">

{{ box_icon("img/AddButton.png", "", "Hinzufügen", width=64) }}

<p>Die Schaltfläche <u>Hinzufügen</u> ermöglicht das Hinzufügen weiterer Fallrohre zur entsprechenden Regenrinne.</p>
