# 🪟 So arbeiten Sie Schritt für Schritt mit Öffnungen

> Das **Öffnungen**-Menü gibt Ihnen die volle Kontrolle über Fenster und andere Arten von Öffnungen in Ihrem Dachprojekt. Sie können neue Öffnungen hinzufügen, deren Größe und Position anpassen und sie sogar mit nur wenigen Klicks duplizieren oder verschieben.
>
> ⚠️ ***Hinweis:** Bestimmte Funktionen wie **Steuerung** und **Bearbeitungs-Schaltflächen** sind nur im **Erweiterten Modus** verfügbar. Lesen Sie die [**Einstellungsanleitung**](13_settings.md)* *für Anweisungen zur Freischaltung aller Funktionen.*

## 1️⃣ Öffnung hinzufügen

> Auf jeder Dachfläche erscheint die **Steuerungsschaltfläche**, sobald Sie das **Öffnungen**-Menü aufrufen. Klicken Sie auf diese Schaltfläche, um die Liste der verfügbaren Öffnungen zu öffnen, wählen Sie den benötigten Typ aus, und dieser wird automatisch auf der gewählten Dachfläche platziert. Natürlich können Sie die Öffnung anschließend weiter anpassen.

## 2️⃣ Öffnungen bearbeiten

> Sobald Sie eine Öffnung im 3D-Modell durch Anklicken auswählen, **können Sie die Öffnung direkt im 3D-Modell durch Ziehen mit der Maus (drag & drop) vergrößern, verkleinern oder verschieben.**
>
> Gleichzeitig öffnet sich das Eigenschaften-Panel der Öffnung. Hier können Sie:

- Breite und Höhe ändern (exakte Werte in Metern eingeben)

- Die Position numerisch für maximale Präzision anpassen

> **👉** Standardfarben von Öffnungen können über die Schaltfläche Standardfarben im Menü auf der linken Seite festgelegt werden. Dort können Sie Standardfarben für ausgewählte Öffnungstypen definieren.

## 3️⃣ Öffnungen kopieren

> **Eine Öffnung fertiggestellt und möchten Sie sie wiederverwenden?**
> Klicken Sie auf die Öffnung und verwenden Sie das Kopieren-Symbol im Eigenschaften-Panel – die ausgewählte Öffnung wird dupliziert, wobei alle Einstellungen erhalten bleiben. So können Sie konsistente Layouts einfach erstellen, ohne von vorne zu beginnen. Passen Sie anschließend einfach die Position der neu erstellten Öffnung im Eigenschaften-Panel an.

## 4️⃣ Öffnungen verschieben

> Es gibt zwei Möglichkeiten, eine Öffnung zu verschieben:

1.  **Drag & Drop** direkt im Modell – einfach anklicken und mit der Maus verschieben.

2.  Nutzen Sie das **Verschieben-Symbol** im Eigenschaften-Panel – nach dem Anklicken erscheint eine Führungslinie und Sie können die Öffnung präziser positionieren.

## 5️⃣ Öffnungen messen

> Mit der Mess-Schaltfläche können Sie die exakte Größe und Position Ihrer Öffnungen im Modell überprüfen.
