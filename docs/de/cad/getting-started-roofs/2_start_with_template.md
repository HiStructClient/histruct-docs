# 🏠 Wie man mit dem Dachmodellieren anhand einer Vorlage beginnt

Wenn Sie ein einfaches Standarddach modellieren möchten und die Maße bereits vorliegen, ist eine der einfachsten Möglichkeiten, unser vordefiniertes Vorlagenangebot zu nutzen:

1.  Beim Erstellen eines neuen Projekts wählen Sie aus den angebotenen vordefinierten Vorlagen eine passende Vorlage aus. Oder öffnen Sie direkt die **Vorlagen** oben links im HiStruct-Menü und wählen Sie eine passende Dachvorlage aus. Die gewählte Vorlage wird direkt in der Anwendung geöffnet.

![Dachvorlagen](img/2Templates.png)

2.  Klicken Sie auf die **Geometrie**-Karte, um die Grundmaße des Gebäudes, die Gebäudehöhe und die Dachneigung festzulegen. Sie können auch die Traufe verschieben, um die Traufkante zu ändern/zu wechseln.

![Geometrie-Schaltfläche](img/2Geometry.png)

> Und das war's! Sie können Ihr Dach jetzt fast fertig sehen. In den nächsten Schritten wählen Sie die Eindeckung und die Anschlüsse, fügen Öffnungen hinzu und generieren die Ausgaben.
