# 🏠 Wie man mit einer Vorlage mit der Dachmodellierung beginnt

Wenn Sie ein einfaches Dach mit Standardform modellieren möchten und die Abmessungen zur Verfügung haben, ist eine der einfachsten Möglichkeiten, Ihr Dachmodell zu erstellen, die Verwendung einer unserer vordefinierten Vorlagen:

1.  Beim Erstellen eines neuen Projekts wählen Sie eine geeignete Vorlage aus den angebotenen vordefinierten Vorlagen aus. Oder öffnen Sie direkt die **Vorlagen** oben links im HiStruct-Menü und wählen Sie eine passende Dachvorlage. Die ausgewählte Vorlage wird direkt in der Anwendung geöffnet.

![Roof templates](img/2Templates.png)

2.  Klicken Sie auf die Karte **Geometrie**, um die Grundabmessungen des Gebäudes, die Gebäudehöhe und die Dachneigung festzulegen. Sie können auch die Traufe verschieben, um die Traufkante zu ändern/umzuschalten.

![Geometry button](img/2Geometry.png)

 Und das war’s! Sie können Ihr Dach nun fast fertig sehen. In den nächsten Schritten wählen Sie die Dacheindeckung, die Verblechung, fügen Öffnungen hinzu und erzeugen Ausgaben.
