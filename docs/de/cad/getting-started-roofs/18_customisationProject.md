# 🔧 HiStruct-Anpassung für Ihr Produkt

Das Installationssystem für Dachzubehör, Anschlüsse, Anschlüsse und das Dachrinnensystem wird für jeden einzelnen Hersteller separat in einer eigenen Konfigurationsdatei eingerichtet. Sie enthält Informationen darüber, wie die Dachfläche mit Zubehör zu belegen ist, mögliche Farbkombinationen, Informationen zur Stückliste (Artikelnummern) und mehr.  
Diese Konfigurationsdateien werden im Rahmen eines Anpassungsprojekts erstellt, bei dem wir den Konfigurator an den Hersteller mit seinem spezifischen Produktangebot anpassen.

## Was passen wir an?

- Konfigurationsdateien für verfügbare Dacheindeckungen, Anschlüsse, Anschlüsse, Dachrinnensystem
- Geometrien einzelner Komponenten, Texturen, Maps bei Bedarf
- Einstellungen für die erfolgreiche Generierung der Stückliste aus der verfügbaren Geometrie, allgemeine Elemente ohne geometrische Darstellung, Befestigungsmittel, Unterlagen usw.
- Ausgaben gemäß den Anforderungen des Herstellers, Stücklisten, Angebote, Zeichnungen, Ansichten, Exporte usw.
- Anbindung an die Preisgestaltung per API an das bestehende System des Herstellers
- Anbindung an das Ökosystem des Herstellers, Angebots-Exporte, Anbindung an ERP und mehr nach Bedarf
