# 🏠 Wie man eine Dachform aus einer Umrisslinie erzeugt & Verwendung einer DXF-Zeichnung für präzises Modellieren

Wenn Sie eine handgezeichnete Skizze haben, können Sie das Dach einfach aus dem Umriss erzeugen. Und wenn Sie sogar eine Vektorzeichnung im **.dxf**-Format besitzen, umso besser – das Einfügen in den Modellierungsbereich erleichtert Ihnen die Arbeit erheblich und Sie sind schon halb am Ziel! Diese Grundlage ermöglicht es Ihnen, Ihr Dach noch genauer zu zeichnen und dank automatischer Fangerfassung mit dem Cursor wird auch die Präzision Ihres Modells verbessert.

**❓Sie haben keine Zeichnung als \*.dxf, sondern nur als PDF-Format? Kein Problem!** Die meisten Strichzeichnungen lassen sich einfach [in \*.dxf umwandeln](4_PDF_to_DXF.md)

🚀 **Legen wir los!**

> Wenn Sie ein neues Projekt erstellen, wählen Sie die Option **Aus Umriss generieren.** ![pokus 1](img/Outlinetepmpl.png) Alternativ können Sie direkt oben links im HiStruct-Menü die **Vorlagen** öffnen und ebenfalls diese Option auswählen. Sie gelangen dann direkt in die Anwendung und springen gleich ins Menü **Geometrie**.
