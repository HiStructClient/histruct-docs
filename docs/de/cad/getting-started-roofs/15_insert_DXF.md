# 👉 Wie füge ich Ihre DXF-Dateien für eine genaue Modellierung ein?

1.  **Wählen Sie die Schaltfläche Unterlage** und klicken Sie auf **Importieren \*.dxf**. Dadurch wird ein Dialogfeld geöffnet, in dem Sie Ihre Zeichnung hochladen können.

2.  **Festlegung des Einfügepunkts.** Nach Auswahl der Datei müssen Sie den Einfügepunkt der Zeichnung angeben, indem Sie in die Szene klicken.

3.  **Einstellung des richtigen Maßstabs der Zeichnung.** Da die Zeichnungen unterschiedliche Maßstäbe haben, muss zunächst der richtige Maßstab eingestellt werden, damit das Modell den tatsächlichen Abmessungen entspricht (Sie können den Maßstab einstellen, indem Sie nach dem Importieren der Datei auf die graue rechteckige Schaltfläche im Modellierbereich klicken):

> ![Maßstab](img/Scale.png)

a.  Klicken Sie zuerst auf die **Maßstab**-Schaltfläche.

b.  Klicken Sie auf zwei Punkte, deren realen Abstand Sie in der Zeichnung kennen.

c.  Geben Sie dann den gewünschten Abstand ein und drücken Sie **Enter**.

d.  Klicken Sie auf **Fortfahren** und die Software berechnet den Maßstab automatisch.

![Maßstab](img/Scale2.png)
>
> **💡Wenn Sie den richtigen Maßstab eingestellt haben, stimmen die gemessenen Werte mit den Linien aus der Zeichnung überein.**

4.  **Das Dach zeichnen.** Klicken Sie einfach auf die Grenzknopf und zeichnen Sie den importierten Dachplan nach.

> Und das war's! Sie können Ihr Dach jetzt fast fertig sehen. Im nächsten Schritt können Sie Trapezblech, Anschlussbleche wählen, Öffnungen anpassen und Ausgaben generieren.
