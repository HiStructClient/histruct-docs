# 👉 Wie fügen Sie Ihre DXF-Dateien für eine präzise Modellierung ein? 

1.  **Wählen Sie die Schaltfläche „Underlay“** und klicken Sie auf **Import \*.dxf**. Dadurch öffnet sich ein Dialogfenster, in dem Sie Ihre Zeichnung hochladen können.

2.  **Festlegen des Einfügepunkts.** Nach der Auswahl der Datei müssen Sie den Einfügepunkt der Zeichnung festlegen, indem Sie in die Szene klicken.

3.  **Einstellen des richtigen Maßstabs der Zeichnung.** Da die Zeichnungen unterschiedliche Maßstäbe haben, ist es zunächst notwendig, den korrekten Maßstab einzustellen, damit das Modell den tatsächlichen Abmessungen entspricht (Sie können den Maßstab einstellen, indem Sie nach dem Import der Datei im Modellierbereich auf die graue rechteckige Schaltfläche klicken):

 ![Scale](img/Scale.png)

a.  Klicken Sie zuerst auf die Schaltfläche **Scale**.

b.  Klicken Sie auf zwei Punkte, für die Sie den realen Abstand in der Zeichnung kennen.

c.  Geben Sie dann den gewünschten Abstand ein und drücken Sie **Enter**.

d.  Klicken Sie auf **Proceed**, und die Software berechnet den Maßstab automatisch.

![Scale](img/Scale2.png)

 **💡Wenn Sie den korrekten Maßstabswert eingestellt haben, stimmen die gemessenen Werte mit den Linien aus der Zeichnung überein.**

4.  **Das Dach zeichnen**. Klicken Sie einfach auf die Schaltfläche „Boundary“ und zeichnen Sie den importierten Dachplan nach.

 Und das war’s! Sie sehen Ihr Dach nun fast fertig. In den nächsten Schritten können Sie die Eindeckung und die Anschlüsse auswählen, Öffnungen anpassen und Ausgaben generieren.
