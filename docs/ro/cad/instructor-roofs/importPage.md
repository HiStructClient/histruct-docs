# Import

Butonul *Import* este utilizat pentru a importa cu ușurință diverse documente pentru a facilita modelarea proiectului dorit. Puteți insera fișiere în format DXF sau OBJ. În plus, aveți opțiunea de a converti documentele din PDF într-unul dintre tipurile de fișiere acceptate, deschizând noi posibilități și facilitându-vă munca de modelare.
