# Personalizarea HiStruct

Sistemul de instalare pentru învelitori acoperișuri, elemente de etanșare și sistemul de jgheaburi este configurat separat pentru fiecare producător specific într-un fișier de configurare dedicat. Acesta include informații despre modul în care zona acoperișului urmează să fie instalată cu învelitoare, combinații posibile de culori, detalii despre lista de materiale (numere de piese) și altele.
Aceste fișiere de configurare sunt create ca parte a unui proiect de personalizare, în cadrul căruia adaptăm configuratorul la producător, cu oferta lor specifică de produse.

## Ce personalizăm?

- Fișiere de configurare pentru învelitori acoperișuri, elemente de etanșare și sistem de jgheaburi
- Geometria componentelor individuale, texturi și hărți, dacă este necesar
- Setări pentru generarea cu succes a listei de materiale, incluzând geometria disponibilă, elemente generale fără reprezentare geometrică, elemente de fixare, straturi de bază etc.
- Ieșiri conform cerințelor producătorului, incluzând liste de materiale, oferte, desene, vederi, exporturi etc.
- Integrare cu sistemul de prețuri prin intermediul API-ului în sistemul existent al producătorului
- Integrare cu ecosistemul producătorului, inclusiv export de oferte, conectare la ERP și funcționalități suplimentare, dacă este nevoie.
