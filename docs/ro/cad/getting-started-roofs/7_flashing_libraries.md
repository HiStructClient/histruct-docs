# 📝 Cum funcționează bibliotecile HiStruct pentru învelitori și tinichigerie

În HiStruct, bibliotecile de învelitori și tinichigerie sunt mai mult decât simple liste de dimensiuni și culori. Ele stochează toate regulile privind modul în care materialele de învelitoare trebuie instalate.

Ca parte a unui [proiect de personalizare](18_customisationProject.md), configurăm biblioteca pentru a include procedurile specifice de instalare pentru materialul de învelitoare al fiecărui producător. Datorită generatorului de variabile încorporat, sistemul aplică apoi aceste proceduri automat, eliminând necesitatea ajustărilor manuale. Acest lucru asigură că învelitoarea este așezată cu acuratețe și eficiență, economisindu-vă timp și menținând modelul precis.
