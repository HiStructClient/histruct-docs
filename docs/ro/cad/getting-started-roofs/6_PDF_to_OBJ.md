# Conversie ușoară a modelului 3D din PDF în OBJ

 Majoritatea dintre noi avem planuri de construcție în PDF sau modele 3D în diverse formate - din software BIM, scanări cu drona sau date satelitare. Dacă aveți un astfel de model 3D în format PDF și doriți să lucrați cu el sau să îl utilizați în HiStruct, va trebui să îl convertiți în format OBJ, care este acceptat în HiStruct. Acest proces nu este dificil și poate fi realizat de aproape oricine.

 **❓Ce formate de model 3D pot converti în OBJ?**

- **Fișiere BIM**: Revit (.rvt), IFC (.ifc), Navisworks (.nwd) 

- **Scanări 3D**: nori de puncte (.las, .laz), plase/mesh-uri (.ply, .stl, .3ds) 

- **Relevări cu drona și satelitare**: de obicei formate mesh (.stl, .pdf) sau ieșiri din software specializat de fotogrammetrie 

- **Formate CAD și de modelare**: .fbx, .dae (Collada), .3ds, .skp (SketchUp) 

 Toate acestea pot fi convertite în OBJ.
