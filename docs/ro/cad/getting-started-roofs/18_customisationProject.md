# 🔧 Personalizarea HiStruct pentru produsul dvs.

Sistemul de instalare pentru accesorii de acoperiș, șorțuri, șorțuri și sistemul de jgheaburi este configurat separat pentru fiecare producător specific, într-un fișier de configurare separat. Acesta conține informații despre cum trebuie acoperită suprafața acoperișului cu accesorii, combinații posibile de culori, informații despre lista de materiale (coduri de piese) și multe altele.
Aceste fișiere de configurare sunt create ca parte a unui proiect de personalizare, în timpul căruia adaptăm configuratorul la producător, cu oferta sa specifică de produse.

## Ce personalizăm?

- Fișiere de configurare pentru învelitori disponibile, șorțuri, șorțuri, sistem de jgheaburi
- geometriile componentelor individuale, texturi, hărți dacă este necesar
- Setări pentru generarea cu succes a listei de materiale din geometria disponibilă, elemente generale fără reprezentare geometrică, elemente de fixare, straturi suport etc.
- ieșiri conform cerințelor producătorului, liste de materiale, oferte, desene, vederi, exporturi etc.
- Conectare la prețuri prin API la sistemul existent al producătorului
- Conectare la ecosistemul producătorului, exporturi de oferte, conectare la ERP și altele, după necesități
