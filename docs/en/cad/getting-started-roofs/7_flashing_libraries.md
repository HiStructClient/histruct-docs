# 📝 How HiStruct Roofing & Flashing Libraries Work

In HiStruct, roofing and flashing libraries are more than just lists of sizes and colours. They store all the rules for how roofing materials should be installed.

As part of a [customisation project](18_customisationProject.md), we configure the library to include the specific installation procedures for each manufacturer's roofing material. Thanks to the built-in variable generator, the system then applies these procedures automatically, eliminating the need for manual adjustments. This ensures the roof covering is laid out accurately and efficiently, saving you time while keeping your model precise.
