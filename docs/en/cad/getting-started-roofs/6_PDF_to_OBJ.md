# Easy Conversion of 3D Model from PDF to OBJ

> Most of us have building plans in PDF or 3D models in various formats - from BIM software, drone scans, or satellite data. If you have such a 3D model in PDF format and want to work with it or use it in HiStruct, you will need to convert it to OBJ format, which is supported in HiStruct. This process is not difficult and can be done by almost anyone.
>
> **❓What 3D model formats can I convert to OBJ?**

- **BIM files**: Revit (.rvt), IFC (.ifc), Navisworks (.nwd) 



- **3D scans**: point clouds (.las, .laz), meshes (.ply, .stl, .3ds) 



- **Drone and satellite surveys**: usually mesh formats (.stl, .pdf) or specialized photogrammetry software outputs 



- **CAD and modeling formats**: .fbx, .dae (Collada), .3ds, .skp (SketchUp) 

> All of these can be converted to OBJ. 

## Conversion using online tools

> There are several online tools that allow you to convert such models. For example:

1.  [Aspose 3D PDF to OBJ Converter](https://products.aspose.app/3d/conversion/pdf-to-obj)

2.  [ImageToSTL PDF to OBJ Converter](https://imagetostl.com/convert/file/pdf/to/obj)

## Conversion using paid applications

> If you need to convert such files on a regular basis, you may consider using paid applications that offer more features and support. These include:

1.  [Tetra4D](https://tetra4d.com/)

2.  [Autodesk Maya](https://www.autodesk.com/products/maya)
